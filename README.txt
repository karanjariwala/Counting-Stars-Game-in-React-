A Pen created at CodePen.io. You can find this one at http://codepen.io/KaranJariwala/pen/MmNQOg.

 
Count the stars by using numbers or sum of numbers.  
Click equal button once to verify the answer, once verified 'Okay', click it again to submit the answer. 

you will get 7 redraws for redrawing the stars randomly.
If you are able to use all the numbers i.e all the numbers are green, you win. If there is no possible solution to count the stars using the available numbers and you have used all the 7 redraws, you lose!
